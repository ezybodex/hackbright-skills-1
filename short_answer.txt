3) Explain the difference between these two commands in the terminal:
   
   $ cd Users
   $ cd /Users

Your answer:

   $ cd Users: this takes you to the Users folder in the directory you're currently. this can only work if there is an existing folder called "Users" in the current directory.
   $ cd /Users: this is on the other hand is an absolute path that directly navigates to the "Users" directory at the root level of the file system.
